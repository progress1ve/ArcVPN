# Как сделать красивое отображение подписок в VPN-клиенте

## Что изменено

Теперь бот использует **название сервера** из базы данных для отображения в VPN-клиенте вместо технического формата `VPN-user_123456`.

## Как это работает

Когда пользователь получает ключ, в VPN-клиенте (v2rayNG, Hiddify, Streisand и т.д.) будет отображаться название сервера, которое вы указали в админ-панели.

## Как добавить эмодзи флагов

### Шаг 1: Откройте админ-панель бота
Нажмите кнопку "⚙️ Админ-панель" → "🖥️ Серверы"

### Шаг 2: Отредактируйте названия серверов
Для каждого сервера добавьте эмодзи флага в начало названия:

**Примеры:**
- `🇺🇸 США`
- `🇷🇺 Россия / Youtube, Discord`
- `🇩🇪 Германия 1`
- `🇩🇪 Германия 2`
- `🇳🇱 Нидерланды`
- `🇫🇮 Финляндия`
- `🇸🇪 Швеция`
- `🇬🇧 Великобритания`
- `🇫🇷 Франция`
- `🇯🇵 Япония`
- `🇸🇬 Сингапур`

### Шаг 3: Сохраните изменения

После этого все новые ключи будут отображаться с красивыми названиями и флагами!

## Важно

- Изменения применяются только к **новым ключам**
- Для обновления существующих ключей пользователь должен использовать функцию "Заменить ключ"
- Название сервера берётся из поля "Название" в настройках сервера в админ-панели бота

## Технические детали

Изменённые файлы:
- `bot/utils/key_generator.py` - функция `_get_remark()` теперь использует `server_name`
- `bot/services/panels/xui.py` - функция `get_client_config()` добавляет `server_name` в конфигурацию
